# Reto Técnico TAE - Automatizador (Serenity + Screenplay + Cucumber)

## Requisitos
- Java 21
- Maven
- Chrome (o cambiar driver en serenity.conf)

## Ejecutar pruebas
1. Abrir terminal en la raíz del proyecto.
2. Ejecutar: `mvn clean verify`

Los reportes se generan en `target/site/serenity/index.html`.
